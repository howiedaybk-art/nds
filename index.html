Инструкция по использованию

  • Введите количество товара и стоимость единицы без НДС в любую строку таблицы
  • Все остальные значения будут рассчитаны автоматически
  • Для добавления новой строки нажмите кнопку "Добавить строку"
  • Для удаления строки нажмите на кнопку с корзиной в конце строки
  • Для расчета итоговых сумм по всем позициям нажмите "Рассчитать итоги"
  • Для очистки всех данных нажмите "Очистить все"
